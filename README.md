# Jeu Dont On Est Le Heros

Ce dépôt contient différents projets .NET issus d'exemples de cours.

## Organisation

- `src/` contient toutes les applications et bibliothèques.
- `tests/` regroupe les projets de tests unitaires.

Le fichier de solution `PratiqueUdemy.sln` référence l'ensemble des projets.

## Utilisation

Les projets ciblent .NET Core 3.1. Pour lancer les tests ou exécuter les applications, utilisez les outils `dotnet` correspondants.
